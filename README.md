## Задание 1: Юнит-тесты

### Автотесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы юнит-тесты, покрывающие класс  `Burger`

Процент покрытия 100% (отчет: `htmlcov/class_index.htm`)

### Структура проекта

- `praktikum` - пакет, содержащий код программы
- `test_burger.py` - пакет, содержащий тесты, разделенные по классам.  ``

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `$ python -m pytest tests/ --cov=praktikum --cov-report=html`
